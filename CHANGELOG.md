# Changelog

所有重要的版本变更记录在此。

## [1.0.0] - 2026-03-26

### 🎉 首次发布

#### 核心功能
- **记录**：随手写下想法，AI 自动整理文字、提炼标签，打卡效率与情绪
- **日记**：月历视图追踪每日记录，支持情绪与效率节奏可视化，关键字搜索日记档案
- **灵光**：AI 周报总结本周心路并给出建议，智能问答随时与过去的自己对话
- **回响**：AI 发现日记中的隐藏关联——成长闭环、情绪解药、行为规律洞察

#### 隐私与安全
- 100% 本地数据存储，所有日记仅存于浏览器 IndexedDB
- 无服务器、无数据上传、无追踪

#### AI 支持
- 多模型接入：DeepSeek / OpenAI / Claude / Gemini / Kimi / 通义千问 / 腾讯混元 / 自定义 API
- 用户自行配置 API Key，密钥仅保存在本地

#### 输入方式
- 键盘文字输入
- 原生 macOS Speech 框架语音转文字

#### 数据管理
- JSON 完整备份与恢复
- Word (.docx) 日记导出

#### 界面
- 浅色 / 深色双主题
- Anthropic Design System 配色体系
